# Skill: Concept Review

## Required Reads

- `concept/00_index.md`
- all relevant concept files
- `concept/09_open_questions.md`
- `decisions/`

## Checks

- terms are used consistently
- no product code is introduced
- MVP scope is not inflated
- permissions are not overpromised
- UI behavior follows capability model
- npm CLI output remains stable for MAUI consumption
- open assumptions are listed

## Allowed Updates

- concept files
- open questions
- ADRs

## Not Allowed

- product code
- package scaffolding
- source folders
